"카페 모임" 프로젝트 초보자용 API 명세 가이드북

OpenAPI란 무엇인가요?

OpenAPI는 프론트엔드와 백엔드가 서로 약속한 규격대로 데이터를 주고받을 수 있도록 작성한 API 계약 문서(Contract)입니다.

💡 이 명세를 기반으로 FastAPI는 실제 작동하는 대화형 테스트 페이지(스웨거: http://localhost:8000/docs)를 자동 생성해 줍니다.

OpenAPI 디렉토리 구성

하나의 거대한 명세서 파일을 사용하는 대신, 성격(역할)별로 분할하여 조합하는 조립식 설계를 사용했습니다.

📁 docs/openapi
├── 📁 components/ # [데이터 규격] 공통 데이터 모델 정의
│ └── 📄 schemas.yaml # UserResponse, PostCreate 등의 JSON 구조체 정의
├── 📁 paths/ # [API 경로] 기능 도메인별 리소스 주소 및 HTTP 메서드 정의
│ ├── 📄 auth.yaml # 회원가입, 로그인, 정보수정, 멤버관리 API 명세
│ ├── 📄 posts.yaml # 자유게시판 글 등록, 조회, 삭제 및 댓글 API 명세
│ └── 📄 events.yaml # 일정 추가, 캘린더 월별 조회, 삭제 API 명세
└── 📄 main.yaml # 🚀 [메인 조립 파일] 분산된 파일들을 하나로 연결하여 명세 완성

구현된 카페 API 목록 (API Endpoint Map)

본 프로젝트에 반영된 주요 API 경로와 사용 권한 요약입니다.

🔐 인증 및 회원 관련

POST /auth/signup 일반 회원 가입 (아이디/닉네임 중복체크 포함)
POST /auth/login 로그인 및 JWT 토큰 발급
GET /auth/me 로그인 유저 본인 프로필 조회
PUT /auth/me 본인 프로필(닉네임, 비밀번호) 수정
GET /auth/users 전체 가입 회원 목록 조회 (관리자 전용)
DELETE /auth/users/{user_id} 회원 강제 탈퇴 처리 (관리자 전용)

📝 자유게시판 및 댓글 관련

GET /posts 게시글 전체 목록 조회
POST /posts 신규 게시글 작성 (로그인 필수)
GET /posts/{post_id} 게시글 상세 정보 및 댓글 리스트 조회
PUT /posts/{post_id} 게시글 수정 (작성자 또는 관리자)
DELETE /posts/{post_id} 게시글 삭제 (작성자 또는 관리자)
POST /posts/{post_id}/comments 댓글 작성
DELETE /posts/{post_id}/comments/{comment_id} 댓글 삭제

📅 공유 캘린더 관련

GET /events 특정 년도/월에 겹치는 일정 범위 조회
POST /events 일정 등록 (종일 일정 여부 및 기간 일정 지원)
DELETE /events/{event_id} 일정 삭제 (작성자 또는 관리자)
