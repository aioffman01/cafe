# 단일 카페 커뮤니티 "카페 모임(Cafe Moim)" 기능 정의서

본 문서는 일반 회원과 관리자가 함께 이용하는 단일 카페 서비스인 **"카페 모임"**의 기능 요구사항을 정의합니다.

---

## 1. 회원 관리 기능 (Member & Authentication)

### 1.1 회원가입 (Sign Up)
- 사용자는 아이디, 비밀번호, 닉네임을 입력하여 회원가입을 요청합니다.
- 비밀번호는 서버에 저장될 때 `bcrypt` 단방향 해시로 암호화됩니다.
- 닉네임과 아이디는 중복될 수 없습니다.

### 1.2 로그인 (Sign In)
- 사용자는 아이디와 비밀번호를 입력하여 로그인을 요청합니다.
- 로그인 성공 시 서버는 사용자 정보(ID, 역할)가 담긴 **JWT Access Token**을 발급합니다.
- 토큰 만료 시간은 기본 60분으로 설정합니다.

### 1.3 내 정보 조회 (Profile)
- 로그인한 사용자는 자신의 프로필 정보(아이디, 닉네임, 역할)를 확인할 수 있습니다.

---

## 2. 게시판 기능 (Board & Posts)

게시판은 전체 회원이 자유롭게 의사소통할 수 있는 공간입니다.

### 2.1 게시글 목록 조회 (Get Post List)
- 작성된 전체 게시글을 최신순으로 조회할 수 있습니다.
- 목록에는 제목, 작성자 닉네임, 작성 시간, 조회수가 표시됩니다.

### 2.2 게시글 상세 조회 (Get Post Detail)
- 특정 게시글을 클릭하면 상세 내용을 확인합니다.
- 본문 내용과 함께 하단에 댓글 목록이 함께 표시됩니다.
- 상세 조회 시 조회수가 1 증가합니다.

### 2.3 게시글 쓰기/수정/삭제 (Write/Edit/Delete Post)
- **쓰기**: 로그인한 회원만 글을 쓸 수 있습니다.
- **수정/삭제**: 자신이 작성한 글만 수정 및 삭제가 가능합니다. 단, **관리자(Admin)**는 타인의 글도 삭제할 수 있습니다.

### 2.4 댓글 쓰기/삭제 (Write/Delete Comment)
- 로그인한 회원은 게시글 상세 화면에서 댓글을 달 수 있습니다.
- 자신이 작성한 댓글만 삭제할 수 있습니다. (관리자는 모든 댓글 삭제 권한을 가집니다.)

---

## 3. 캘린더 기능 (Calendar & Events)

캘린더는 단체 일정, 소모임 일정, 공식 행사를 한눈에 보여주는 공유 캘린더입니다.

### 3.1 월별 일정 조회 (Get Monthly Events)
- 달력(Calendar) UI 상에서 특정 연도/월에 해당하는 모든 일정을 불러와 시각적으로 표시합니다.
- 관리자가 작성한 공식 일정과 일반 회원이 작성한 친목 일정을 구분하여 표시합니다.

### 3.2 일정 등록 (Create Event)
- 로그인한 회원은 누구나 일정을 등록할 수 있습니다.
- 입력 정보: 일정 제목, 설명, 시작 날짜/시간, 종료 날짜/시간
- **중요**: 관리자 계정으로 일정을 등록하면 해당 일정에는 자동으로 `is_admin_event: true` 값이 설정되며, 달력 UI 상에서 강조(예: 디자인 가이드에 따른 Red 포인트 테두리 및 배지) 표시됩니다. 일반 회원 일정은 Blue 테두리로 표시됩니다.

### 3.3 일정 수정/삭제 (Edit/Delete Event)
- 자신이 등록한 일정만 수정 및 삭제가 가능합니다. 관리자는 모든 일정을 관리(수정/삭제)할 수 있습니다.
